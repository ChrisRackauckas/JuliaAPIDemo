DifferentialEquations.jl Online

This page allows you to use some of the functionality of DifferentialEquations.jl in your browser. To get started, click on one of the tools above.

Authors

This tool was built by Alex Mellnik and Chris Rackauckas.

How it works

DifferentialEquations.jl is a package written in Julia, a high-performance language for technical computing. A web interface frontend was built using Angular and was linked to Julia via Mux.jl. The results are parsed via ParameterizedFunctions.jl, a component of the DifferentialEquations.jl ecosystem which performs conversions between natrual syntax to Julia code for parameterized differential equations. The end result is a clean interface for users to explore some of the functionality of a performant Julia library directly from their browser.
